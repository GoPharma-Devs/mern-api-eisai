## Packages that are used 📦

- express
- cors
- dotenv
- helmet
- mongoose
- bcrypt
- jsonwebtoken
- nodemailer

### Dev Dependencies 👩🏾‍💻

- nodemon
- morgan

## Instalation 🐝

```bash
npm install
```

### Run on dev mode 🔍

```bash
npm run dev
```

### Start production mode 📽

```bash
npm start
```
